Multi-Document Summarizer: CS535 Term Project

Project Info

We implement an abstractive multi-document summarization pipeline over the Multi-News dataset using three state-of-the-art seq2seq models: Pegasus-Large, BART-Large-CNN, and LED-Base-16384. Our backend is powered by PyTorch & DDP.

Usage: Select one or more models, add your documents (or pick random examples), then hit “Summarize” to see each model’s output side by side.
